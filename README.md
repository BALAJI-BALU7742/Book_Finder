# Book Finder App

A React app to search books and manage favorites with favorites toggle.

## Features
- Search books with debounce
- Favorites toggle
- Expandable search bar
- Responsive design
- Modal for book details

## Tech Stack
- React
- React Icons
- CSS

## How to Run Locally
1. `npm install`
2. `npm start`
